=========
raster
=========

The SlideRule Python API ``raster.py`` is used to sample and (in the future) subset raster datasets registered with SlideRule. See `GeoRaster <../user_guide/GeoRaster.html>`_ for more details. From Python, the module can be imported via:

.. code-block:: python

    from sliderule import raster
